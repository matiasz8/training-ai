# Práctica 01 — Python y NumPy desde cero

## Objetivo
Dominar operaciones básicas con arrays para preparar datos numéricos.

## Enunciado
1. Crear un array de 20 números enteros.
2. Calcular suma, promedio, mínimo y máximo.
3. Normalizar el array al rango [0, 1].
4. Mostrar los 5 valores más altos.

## Entregable
- Script o notebook con resultados impresos.
- Breve explicación de cada operación.

## Validación
- El código corre sin errores.
- Se usan operaciones vectorizadas (sin loops innecesarios).
- Resultados consistentes y explicados.

## Errores comunes
- Mezclar listas de Python con arrays sin convertir.
- Dividir por cero al normalizar.

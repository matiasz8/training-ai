# Training AI — de 0 a senior (en español)

Este training está diseñado para estudiar todos los temas de IA de forma secuencial, práctica y progresiva.

## Empezar por acá (paso a paso)
1. Abrí la guía central de documentación: `docs/README.md`.
2. Leé el camino de estudio sin fechas fijas: `docs/LEARNING-PATH.md`.
3. Revisá qué vas a aprender en todo el recorrido: `docs/RESUMEN-MODULOS.md`.
4. Empezá por el módulo `modules/01-programming-math-for-ml/`.
5. Dentro de cada módulo seguí siempre este orden: `theory` → `examples` → `practices` → `mini-project` → `evaluation`.
6. Marcá avances y bloqueos en `STATUS.md` del módulo antes de pasar al siguiente.

## Objetivo general
- Cubrir los 12 temas principales de `training-ai` sin omitir ninguno.
- Desarrollar confianza técnica con evidencia práctica real.
- Avanzar desde fundamentos hasta decisiones de nivel senior.

## Principios de estudio
- Todo el contenido se trabaja desde nivel inicial.
- Todas las explicaciones y documentación se mantienen en español.
- Cada módulo incluye teoría, ejemplos, práctica, mini-proyecto y evaluación.
- Se trabaja con foco en un módulo a la vez.

## Estructura
- `modules/`: módulos temáticos numerados.
- `templates/`: plantillas reutilizables para prácticas/proyectos/evaluación.
- `shared/`: recursos compartidos (datasets, utilidades, notebooks).
- `trends-extras/`: tendencias nuevas para ampliar criterio técnico.
- `docs/`: documentación centralizada del training.

## Nota sobre `init-path/`
- `init-path/` se conserva como referencia histórica del primer ejemplo.
- El recorrido vigente y mantenido está en `modules/` + `docs/`.

## Documentos clave
- `docs/README.md`: índice de documentación.
- `docs/LEARNING-PATH.md`: ruta de estudio por fases y pasos (sin semanas).
- `docs/RESUMEN-MODULOS.md`: resumen centralizado de qué se aprende en los 12 módulos.
- `docs/PHASES-CREACION.md`: fases de implementación del training.
- `docs/IMPLEMENTATION-STATUS.md`: estado de construcción del repositorio.
- `docs/STUDY-RHYTHM.md`: ritmo sugerido adaptable.
- `docs/RESOURCES.md`: recursos base recomendados.

# Programación y Matemática para ML

## Objetivo del módulo
Construir una base sólida desde cero en Python y matemática aplicada para entender, entrenar y mejorar modelos de machine learning.

## Qué vas a lograr
- Entender álgebra lineal, cálculo y probabilidad con intuición práctica.
- Usar Python (NumPy, Pandas, Matplotlib) para resolver problemas de datos.
- Implementar operaciones matemáticas y métricas con código reproducible.
- Completar un mini-proyecto de regresión lineal desde cero.

## Estructura interna
- theory/: fundamentos en español.
- examples/: ejemplos guiados.
- practices/: ejercicios por dificultad.
- mini-project/: caso integrador corto.
- evaluation/: checklist y rúbrica.
- notes/: apuntes y decisiones técnicas.

## Plan recomendado (4 semanas)
- Semana 1: teoría base + glosario + operaciones básicas en Python.
- Semana 2: ejemplos guiados de vectores, matrices, derivadas y distribuciones.
- Semana 3: prácticas 01 a 05 + correcciones.
- Semana 4: mini-proyecto + evaluación final.

## Ruta de niveles (L1-L4)
- L1 Fundamentos: variables, funciones, arrays, álgebra lineal básica.
- L2 Aplicado: derivadas, optimización, probabilidad y estadística descriptiva.
- L3 Avanzado: implementación de gradiente descendente y validación de resultados.
- L4 Senior: análisis de trade-offs numéricos, reproducibilidad y calidad experimental.

# Deep Learning Basics

## Objetivo del módulo
Entender los fundamentos de deep learning para construir, entrenar y evaluar redes neuronales simples con criterio técnico.

## Qué vas a lograr
- Comprender neuronas artificiales, capas y funciones de activación.
- Entrenar una red neuronal básica para clasificación.
- Identificar overfitting y aplicar regularización.
- Evaluar modelos con métricas y curvas de aprendizaje.

## Estructura interna
- theory/: fundamentos en español.
- examples/: ejemplos guiados.
- practices/: ejercicios por dificultad.
- mini-project/: caso integrador corto.
- evaluation/: checklist y rúbrica.
- notes/: apuntes y decisiones técnicas.

## Ruta de niveles (L1-L4)
- L1 Fundamentos: arquitectura básica de redes y forward pass.
- L2 Aplicado: entrenamiento con backpropagation en frameworks modernos.
- L3 Avanzado: regularización, tuning y debugging de entrenamiento.
- L4 Senior: decisiones de arquitectura por costo, precisión y tiempo.

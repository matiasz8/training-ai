# Theory — Generative AI & Prompt Engineering

## 1) ¿Qué es IA generativa?
Conjunto de modelos capaces de crear contenido nuevo (texto, imagen, código, audio) a partir de patrones aprendidos.

## 2) Prompt engineering
Es el proceso de diseñar instrucciones para obtener salidas más precisas y útiles.

## 3) Estructura de un buen prompt
- Rol/contexto.
- Tarea concreta.
- Restricciones.
- Formato de salida.
- Criterio de calidad.

## 4) Técnicas básicas
- Zero-shot.
- Few-shot.
- Chain of thought guiado (cuando aplica).
- Refinamiento iterativo.

## 5) Riesgos frecuentes
- Alucinaciones.
- Respuestas inconsistentes.
- Sesgos.
- Exposición de datos sensibles.

## 6) Guardrails básicos
- Validaciones de formato.
- Filtros de contenido sensible.
- Límites de alcance de respuesta.
- Verificación con fuentes externas cuando aplique.

## 7) Evaluación de calidad
- Correctitud.
- Relevancia.
- Coherencia.
- Seguridad.

## 8) Checklist de comprensión
- ¿Podés convertir una necesidad de negocio en prompt claro?
- ¿Sabés iterar un prompt frente a una respuesta mala?
- ¿Podés definir métricas de evaluación para respuestas generadas?

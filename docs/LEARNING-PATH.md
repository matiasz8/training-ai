# Learning Path (por progreso, no por calendario)

Avance secuencial por módulo. Tu ritmo lo definís vos; el criterio para avanzar es completar resultados, no semanas.

## Orden recomendado de módulos

### Fase 1 — Fundamentos
1. Programación y matemática para ML
2. Data collection, cleaning & visualization
3. Machine learning fundamentals

### Fase 2 — Modelado moderno
4. Deep learning basics
5. NLP & large language models
6. Computer vision

### Fase 3 — Especializaciones
7. Time series & anomaly detection
8. Recommender systems
9. Generative AI & prompt engineering

### Fase 4 — Producción y responsabilidad
10. Ethics, bias & explainability
11. Data privacy & security
12. MLOps & AI in production

## Paso a paso dentro de cada módulo
1. Leer `README.md` del módulo para entender objetivo y alcance.
2. Estudiar `theory/README.md` y registrar síntesis en `notes/`.
3. Ejecutar `examples/` para entender flujo base.
4. Resolver prácticas de `practices/` en orden.
5. Completar `mini-project/` con resultados reproducibles.
6. Cerrar `evaluation/` y verificar mínimo 70/100.
7. Actualizar `STATUS.md` (nivel, bloqueos, próximos pasos).

## Regla para pasar al siguiente módulo
No avanzar hasta cumplir:
- Teoría resumida.
- Prácticas completas.
- Mini-proyecto validado.
- Evaluación aprobada.
- Lecciones aprendidas documentadas.
